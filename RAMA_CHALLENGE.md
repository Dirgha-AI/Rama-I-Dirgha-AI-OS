# Rama I — The Sovereign Computer Challenge

## Building India's First Open-Source Computer

**The Problem:** India assembles 25% of the world's iPhones, but the "intelligence"—the chips, the OS, and the IP—is owned elsewhere.

**The Solution:** Rama I. A national open challenge to build, document, and ship a working computer using India's own companies and open-source designs.

---

## 🛠️ The Vision

Rama I is about building the foundation. Designing it, building it, owning it. We are building the "IKEA instructions" for sovereign hardware.

### The Target Stack\*
Rama I aims to pull together existing Indian hardware and software capability into a single, cohesive, open-source device. The table below is the **target architecture** — the stack we are scoping against as we close supplier agreements.

| Layer | Source | Status |
|---|---|---|
| **SoC / Brain**\* | C-DAC / IIT Madras | RISC-V (VEGA / SHAKTI) |
| **PCB Fab**\* | Bengaluru cluster | Domestic fabrication |
| **Assembly**\* | NCR cluster | Domestic SMT assembly |
| **Power**\* | Chennai cluster | Domestic manufacturing |
| **OS / Firmware** | Open Source | Linux + OpenSBI |

### Why RISC-V?
RISC-V is an open-source instruction set architecture (ISA). It is the global standard for sovereign computing, and India has bet heavily on it through the **Digital India RISC-V (DIR-V)** initiative. Rama I will be built on this foundation.

## 📊 The DVA Tracker (Domestic Value Addition)\*

We are committed to total transparency. The target is that every Rama I unit ships with a verifiable record of its "Indian-ness."

**Target DVA: 75% by 2029**
**Current status:** target BOM defined, supplier conversations in progress, verification framework being designed.

| Component | Supplier conversations under way with\* |
|---|---|
| **SoC / Brain** | C-DAC (VEGA design, Taiwan fabrication), IIT Madras (SHAKTI) |
| **PCB Fab** | Bengaluru PCB fabrication cluster |
| **Assembly** | NCR / Manesar SMT cluster |
| **Enclosure** | Pune CNC cluster |
| **Power** | Chennai power-supply manufacturers |
| **Memory** | Micron India (assembly-only today) |
| **OS / Firmware** | Open Source (Linux + RISC-V OpenSBI) — no supplier needed |

The DVA percentages and named vendor commitments will be published in a signed BOM manifest when Phase 0 lock is complete. Until then, consider this table directional — an invitation to the community to help close the memory and silicon-fabrication gaps, not a record of executed agreements.

## 🚀 The Roadmap

We are executing on a multi-phase, vertically integrated roadmap:

1.  **Phase 0:** Freeze the BOM (Bill of Materials). Use existing Indian-designed chips (like VEGA/SHAKTI) and domestic PCB/assembly partners.
2.  **Phase 1:** Firmware and OS sprint. Port the Dirgha AI OS to the sovereign hardware stack.
3.  **Phase 2:** Launch the first prototypes for education and developer communities.
4.  **Phase 3:** Scale the "factory in every city" model for sovereign compute.
