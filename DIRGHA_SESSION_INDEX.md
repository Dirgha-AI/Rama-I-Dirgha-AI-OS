# Dirgha AI OS: Session Index
**Date:** April 22, 2026  
**Mission:** Accelerate Abundance

This document serves as the central index for all technical, strategic, and biographical documentation generated during this session for the launch of the [**Rama-I-Dirgha-AI-OS**](https://github.com/Dirgha-AI/Rama-I-Dirgha-AI-OS) repository under the [Dirgha-AI GitHub organization](https://github.com/Dirgha-AI).

> **Reading convention.** Items marked with an asterisk (`*`) across these documents are on the committed roadmap and in active development — they reflect the full Dirgha vision but are not yet shipping in production. What ships today runs at [dirgha.ai/app](https://dirgha.ai/app): Chat, IDE (Monaco + ghost text + Composer), Writer, Research, Library, Marketplace, Creator, Education, and Manufacturing workspaces, backed by a live TypeScript/Hono gateway, Python/LangGraph agent-server, and `@dirgha/cli` beta.
