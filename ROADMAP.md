# Roadmap: The Path to Abundance

## Vertically Integrated AI & Physical Infrastructure

Dirgha is executing on a multi-phase roadmap to build a fully sovereign, open-source stack for the agentic era.

### Phase 0: Foundations & Private Build (Current — April 2026)
**Goal:** Develop core technologies and ready the initial product suite for public launch.
*   **Platform Live:** Chat, IDE (Monaco + inline ghost text + Composer), Writer, Research, Library, Marketplace, Creator, Education, and Manufacturing workspaces shipping at [dirgha.ai/app](https://dirgha.ai/app).
*   **Gateway + Agent-Server:** 417-route TypeScript/Hono gateway + Python/LangGraph agent-server with sandbox (E2B, Daytona), repo-map, deep research.
*   **Billing Live:** Stripe + Razorpay subscriptions, `@dirgha/pricing` canonical rate card, usage-based metering with platform markup.
*   **CLI:** `@dirgha/cli` shipping as a 4.4 MB ESM bundle (beta); tsc --strict clean.
*   **Abundance Protocol:**\* Developing the P2P compute mesh for distributed inference.
*   **Arniko Security:** Building the first automated red-teaming and security agent.
*   **Dirgha Code:** Advance autonomous software engineering and agentic code generation.
*   **Workspace Studio:** Preparing Writer and Creator\* studios for AI-native production.
*   **Rama I:**\* Scoping the initial bill of materials (BOM) for India's first open-source computer.

### Phase 1: Public Launch & Ecosystem Scaling
**Goal:** Transition repositories to public access and scale the decentralized economy.
*   **Open Access:** Opening our core software repositories under the [Dirgha-AI GitHub organization](https://github.com/Dirgha-AI) for community contribution.
*   **Abundance Marketplace:**\* Enable global companies to post work and local guilds to execute it autonomously.
*   **Bucky Mesh Expansion:**\* Grow the decentralized compute network of GPU nodes.
*   **Rama I Prototype:**\* Reveal and iterate on the first physical prototypes of the Rama I computer.

### Phase 2:\* Robotics & Physical Infrastructure
**Goal:** Extend the OS into the physical world and automate production.
*   **City-Factory Model:**\* Pilot the first AI-optimized, open-process manufacturing blueprint for Dirgha hardware.
*   **Robotics Workflow Orchestration:**\* Integrate robots directly into the AIOS Kernel for manufacturing and logistics.
*   **Custom SoCs:**\* Advance toward custom Dirgha-designed RISC-V System-on-Chip (SoC) using AI-assisted design tools.

### Phase 3:\* Global Sovereign Compute & Beyond
**Goal:** Sovereign compute for every nation.
*   **Global Blueprint:**\* Open-source the manufacturing blueprints so any city or nation can build its own sovereign compute infrastructure.
*   **Next-Generation Compute:**\* Invest in photonic and biological compute paradigms through research partnerships.
*   **Autonomous Aerospace:**\* Provide the computational and operational backbone for future aerospace and autonomous logistics infrastructure.

## 🏗️ Technical Development Pillars

1.  **Vertical Integration:** Controlling the full stack—from the P2P protocol to the AIOS Kernel and the RISC-V hardware.
2.  **Decentralization:** Ensuring that no single entity owns the foundational tools of the new economy.
3.  **Security-First:** Every action in the agentic OS is audited and sandboxed.

**The mission is to accelerate abundance. The stack is being built.**
